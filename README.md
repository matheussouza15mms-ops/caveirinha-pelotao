# caveirinha-pelotao

Aplicacao web para gestao do efetivo e dados administrativos da 1a Cia Fuz, com backend em Supabase e integracao com Google Sheets para mapa da forca diario.

## Estado Atual
- Login e sessao reais via Supabase Auth.
- Quadro Organizacional, Efetivo, FO, Historico/Obs, Punicoes, TAF e TAT integrados ao Supabase.
- Controle Sanitario com dashboard, contagem por regra de negocio, filtro por perfil/pelotao e Ficha Medica por militar.
- Tela de Configuracoes integrada ao app, com nome do usuario, foto local, tema claro/escuro, personalizacao de cor e fluxo de relatar problema via WhatsApp.
- Cabecalho com saudacao dinamica por horario e nome do usuario configurado.
