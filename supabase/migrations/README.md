# Migrations

Crie aqui apenas mudancas incrementais do banco.

Formato sugerido:
- `YYYYMMDDHHMMSS_descricao_curta.sql`

Fluxo:
1. Criar a migration.
2. Revisar o impacto no app.
3. Executar pelo SQLTools no banco remoto.
4. Manter o arquivo versionado no projeto.

Observacao:
- esta pasta deve ficar vazia ate existir uma mudanca real a aplicar no remoto.
