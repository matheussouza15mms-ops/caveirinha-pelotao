# Projeto: Caveirinha App

## Objetivo
Aplicação web para gerenciamento do Quadro Organizacional e controle de Efetivo de um pelotão.

## Stack Atual
- HTML5
- CSS3
- JavaScript puro (Vanilla JS)
- Sem backend
- Sem banco de dados persistente

## Estrutura de Arquivos
- index.html
- styles.css
- app.js
- assets/
   - icons/
   - logo-pelotao.png

## Telas Existentes
1. Quadro Organizacional
2. Efetivo
3. Ficha do Militar

## Estado Atual
- Sidebar retrátil funcional
- Sistema de abas verticais por seção
- Cards animados
- Busca lateral com painel flutuante
- Controle de efetivo com KPIs dinâmicos
- Navegação entre telas funcionando

## Fonte Oficial da Verdade
O estado oficial do projeto é sempre o último commit válido no Git.
