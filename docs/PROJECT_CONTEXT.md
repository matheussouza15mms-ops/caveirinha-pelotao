# Projeto: Caveirinha App

## Objetivo
Aplicação web para gerenciamento do Quadro Organizacional e controle de Efetivo de um pelotão, preparada para futura integração bidirecional com Google Sheets via Google Apps Script.

## Stack Atual
- HTML5
- CSS3
- JavaScript puro (Vanilla JS)
- Sem framework
- API layer local (simulação de backend)

## Estrutura de Arquivos (estado atual)
- `index.html`
- `styles.css`
- `app.js`
- `services/api.js`
- `mock/db.json`
- `assets/`
  - `icons/`
  - `logo-pelotao.png`
- `docs/`

## Telas Existentes
1. Quadro Organizacional
2. Efetivo
3. Ficha do Militar

## Estado Atual
- Sidebar retrátil funcional
- Sistema de abas verticais por seção
- Cards animados
- Busca lateral com painel flutuante
- Controle de efetivo com KPIs dinâmicos
- Navegação entre telas funcionando
- Dados carregados via camada de API (`services/api.js`)
- Mock local em JSON (`mock/db.json`) com estrutura compatível com o modelo futuro do Sheets

O estado oficial do projeto é sempre o último commit válido no Git.
